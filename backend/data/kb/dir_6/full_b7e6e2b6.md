本课不少地方运用了排比句，增加了新鲜感。排比是把三个或三个以上意义相关或相近，结构相同或相似，语气一致的词组或句子排列起来，达到了加强语势的效果。

6.“早晨，从山坡上，从坪坝里，从一条条开着绒球花和太阳花的小路上，走来了许多小学生，有汉族的、有傣族的，有景颇族的，有阿昌族和德昂族的。”这句话告诉我们这些来自不同民族的小朋友，早早地起床，迎着朝阳，踩着露珠，高高兴兴地朝着一个共同一个地方─ 学校走去。

# 我的同学

“哈哈哈……”一阵笑声从三年级七班的教室里传来，你知道这是谁的笑声吗？

她长着长头发，穿着黑色的鞋子，个子不高，有一双明亮的大眼睛，有着非常大的嗓门，她如果喊一声，就能震翻整个三年级七班，像一只 “大老虎”。

她很爱笑，别人随随便便说个冷笑话就能把她逗笑。我跟她恰恰相反，即使说个好笑的笑话，也未必能把我逗笑。

她学习不错，每次考试都能考出很好的成绩，是同学们学习的榜样。记得一次，老师讲了“三位数除以一位数”的竖式后，留下了几道有点难度的练习题。前几道，我做得比较顺利。可是，临到最后一道时，就被“卡”住了。我左思右想，绞尽脑汁也没做出来。正当我要放弃时，她走过来了，仿佛看透了我的心思，便拿起纸笔耐心地给我讲解起来。她讲得那么仔细，思路那么清晰，每讲一步都要和我交流，生怕我听不懂。终于，在她的帮助下，我顺利解出了这道题，心里别提多高兴了。

她爱帮助同学。有一次，我上美术课没带水彩笔，她就好心地把水彩笔借给了我，说：“我们一起用吧！”听了她的话，我很感激她。

# 4《古诗三首》

1. 《山行》是唐代诗人杜牧写的，全诗如下：远上寒山石径斜，白云深处有人家。停车坐爱枫林晚，霜叶红于二月花。诗中“生”的意思是产生，生出，形象地表现了白云升腾、缭绕和漂浮的种种动态；“坐”的意思是因为，霜叶的红艳胜过春天的鲜花，让原本萧条的秋景变得生机勃勃，这也是诗人停车欣赏的原因。

2. “一年好景君须记，最是橙黄橘绿时。”这两句话选自宋代苏轼的《赠刘景文》诗人认为一年中景色最好的时候是秋末冬初。以橙黄橘绿写出了秋天硕果累累的丰收景象，显露出勃勃生机，也是对身处困境中的刘景文的劝勉。人到壮年，虽已青春流逝，但也是人生成熟、大有作为的黄金阶段，勉励朋友珍惜这大好时光，乐观向上、努力不懈，切不要意志消沉、妄自菲薄。“荷尽已无擎雨盖，菊残犹有傲霜枝。”“擎”意思是举。用傲霜斗寒的菊花表达了对友人坚贞、高洁品格的称颂。诗虽为赠刘景文而作，所咏却是深秋景物，无一字涉及刘氏本人的道德文章，这似乎不是题中应有之义，但实际上，作者的高明之处正在于将对刘氏品格和节操的称颂。不着痕迹地糅合在对秋末初冬景物的描写中。

3. 《夜书所见》中“书”的意思是书写。“萧萧梧叶送寒声，江上秋风动客情。”“萧萧”反衬了秋夜的寂静。“寒”一语双关，既写秋风寒冷，又写诗人漂泊天涯的凄凉。这里的“客”指的是漂泊在外的宋代诗人叶绍翁，这里的“情”指的是思乡之情。“知有儿童挑促织，夜深篱落一灯明。”的场景诗人倍感亲切，更显出他客居他乡的孤寂与无奈。“促织”指的是蟋蟀，“篱落”指的是篱笆。

4. 多音字：挑 tiāo 挑选、挑担 tiǎo 挑战、挑拨

巧记：她自信满满地挑（tiāo）选了一个具有挑（tiǎo）战性的比赛项目。

停车坐爱枫林晚：因为。 夜书所见：书写。 知有儿童挑促织：蟋蟀。 白云生处有人家：产生，生出。

6. 《山行》《赠刘景文》《夜书所见》这三首诗描绘的都是秋天的景色。从“寒山”和“霜叶”这两个词不难看出《山行》写的就是秋天的景色。《赠刘景文》“橙黄橘绿”写出了秋天硕果累累的丰收景象，“荷尽已无擎雨盖，菊残犹有傲霜枝。”中的“荷尽”和“菊残”都点明是秋天。《夜书所见》梧叶摇落、江上秋风烘托了深秋寒凉的气氛。“萧萧梧叶送寒声，江上秋风动客情。”中“秋风”明显点出是秋天。

“一年好景君须记，正是橙黄橘绿时。”一年中最好的景致你一定要记住，最美的是在秋末初冬橙黄桔绿的时节啊。这里的“君”是诗人对好友刘景文的尊称，相当于现在的“您”。

# 习作2《写日记》

# 关键 1：“写日记的好处”

小朋友，你写过日记吗？把自己当天看到的、听到的，想到的人、事、物和做过的事按照日记既定的格式写下来，就是写日记。不管时间过去多久，不管日月如何变化，不管世界如何发展，我们都可以用日记记录自己生活的所见、所闻、所感和所做。翻开日记本，生活的点点滴滴，成长的喜怒哀乐就会如电影一样一幕幕浮现，这是多么有意义而有趣的事呀！写日记，可以督促我们留心观察生活，也是积累习作素材，进行作文练笔的好方法，经常认真地写日记，就不愁写不好作文了。养成写日记的习惯，是很多有成就的人的共同特点。

吃过早饭后，妈妈就推着自行车带我到公园去学骑车。我鼓起勇气跨上自行车，两手紧紧地握住车把，小心脏“咚咚”乱跳，紧张极了。不过，想到有妈妈的陪伴，我悬着的心又放下了一半。开始学骑车了！妈妈扶着自行车的后面，让我把左脚放在踏板上，右脚放在地上。我用右脚使劲一蹬，车子就慢慢向前移动了。就这样，我生硬地踏着脚踏板往前移动，妈妈在后面紧紧地追随着我。练了一会儿，妈妈已经满头大汗，我很心疼，想让她休息一会儿，就自己试着慢慢来。

骑着骑着，我逐渐加快了速度。不好！前面怎么突然冒出了一块大石头？我的自行车一歪，整个人重重地摔在地上。顿时，我浑身疼痛，泪水控制不住地在眼眶里打转转。那个瞬间，我甚至都想放弃了。这时，妈妈赶紧跑过来，拍着我的肩说：“没关系，练习中摔倒是常见的事，我的女儿平时可是个‘女汉子’啊！”听了妈妈的话，我鼓足了劲继续练习。不知不觉中，我竟然学会骑自行车了！骑在自行车上，我感觉自己就像一只快乐的小鸟，自由地飞来飞去！我要谢谢妈妈，原来，勇敢和坚强真的可以创造奇迹！

# 8《卖火柴的小女孩》

1. 《卖火柴的小女孩》是一篇童话故事，作者是丹麦作家安徒生。他被称为“世界童话大王”，主要作品有《丑小鸭》《皇帝的新装》《夜莺》《海的女儿》，这篇课文讲了一个卖火柴的小女孩在大年夜冻死街头的故事，表达了作者对穷苦人民悲惨遭遇的同情以及对统治者的不满，深刻地揭示了社会的罪恶。  
2. 《卖火柴的小女孩》中，小女孩一共擦亮了五次火柴，第一次看到了暖和的火炉，因为她很冷，渴望得到温暖；第二次看到了喷香的烤鹅，因为她饥饿，渴望得到食物；第三次看到了美丽的圣诞树，因为她孤独，渴望得到快乐；第四次看到了慈爱的奶奶；第五次看到了高大、美丽的奶奶，因为她痛苦，渴望得到亲人的疼爱。最后小女孩的命运是冻死在街头。

3. 多音字：

巧记：明明几（jī）乎每天都要朗读几（jǐ）首诗。

巧记：飞蛾拼命挣（zhēng）扎，也没能挣（zhèng）脱蜘蛛网，眼睁睁地成了蜘蛛的晚餐。

5. 本文最大的写作特色是虚实结合，作者由现实展开丰富的想象，五次幻想将小女孩的悲惨表现到极致。幻想是那么美好幸福，现实又是那样残酷痛苦，加深了作品的感染力。

# 习作3：我来编童话

# 关键 1：“故事里有哪些角色”

课本中提供了“国王”“啄木鸟”和“玫瑰花”这三个角色，你最喜欢谁呢？选择其中的一个或两个角色编故事。当然，你可能有更喜欢的其他角色，比如，小公主、月亮、熊猫、一枚鸡蛋、一群星星……你可以选择一些添加到故事中来，有了可爱生动的角色，精彩的童话剧就快上演了。

# 关键 2：“事情发生在什么时间？是在哪里发生的”

选定了故事的角色，接下来要思考故事发生在什么时间？是在哪里发生的？这两个问题往往是连在一起思考的，比如，选择了“国王”这个角色，那就想象一下，关于国王的事情会发生在“黄昏”“冬天”，还是“星期天”，或是都有可能。事情发生在哪儿呢？“厨房”“森林超市”还是 “小河边”？你可以拿起笔在角色、时间、地点这三者之间连起一条线，一个有趣的故事就这样开始了。

# 关键 3：“他们之间发生了什么故事”

怎样使自己编的童话故事好看呢？那就要让故事的情节丰富起来。我们不仅要思考故事的发生，还要想象故事的发展，再想个有意思的结尾，并且要注意故事前 后之间的联系，不能写好前面就不顾后面的，也不能写着后面的忘记前面的。我们也可以写出角色之间的对话，让角色更生动，让故事更具

体、更有趣。

# 长颈鹿的长脖子

长颈鹿有一条长脖子，像竹竿似的耸立着。

这一天，长颈鹿正在路上散步，遇到了小马。

小马说：“咦，长颈鹿啊，你的脖子怎么这么长啊？你看我的脖子，短短的，你的脖子一定是生病了。”

后来，长颈鹿又遇见了小鹿、小兔子，甚至连啄木 鸟都要议论它的长脖子，长颈鹿不安起来。

它自言自语地说：“它们都这么说，难道我的脖子 真的生病了？我得让脖子缩起来。”于是，长颈鹿把脖 子缩成了“o”形。

可是过了没多久，它就犯难了，因为这样既啃不到叶子，又吃不到小草。

突然，长颈鹿听到了一阵哭声。它抬头一看，原来是一个小朋友被困在了树上。它连忙走过去，把缩着的脖子伸长了，用自己的长脖子当滑梯，把小朋友救了下来。

长颈鹿想：我还是把脖子伸直吧。人家是人家，我是我。长脖子也有长脖子的用处啊！

# 13\*《胡萝卜先生的长胡子》

2. 知名人物名字的故事：

①冰心，也叫谢婉莹，福建长乐人，中国著名的现代诗人和作家。冰心的这个名字，来源于唐朝王昌龄的诗“洛阳亲友如相问，一片冰心在玉壶”，冰心她自己的解释，首先冰心两个字和她名字的莹对应，而且简单好记。其次她说自己胆子比较小，一开始害怕用真名被人批评笑话，就取了冰心这个笔名，没人认识。

②曹雪芹，清代学者，四大名著之一《红楼梦》作者。曹雪芹生性好酒，喜欢美食，据说他最喜欢的一道菜叫做“雪底芹菜”，家道中落后，曹雪芹就经常自己动手做这道菜，这道菜是由雪底的芹菜嫩芽和肉丝炒成。后来干脆就取雪芹两个字作为自己的名字，这就是曹雪芹名字的来历，从一道名菜中得来。  
③李白名字的由来，也很有意思。李白的父母在给李白取名的时候，非常谨慎，一直以来都没有正式的名字。直到在李白七岁的时候，李白的父母要给李白取一个比较正式的名字。一个春天，李白的父母想要考考李白的能力，他的父亲看到庭院花开艳丽，就想作一首四绝。李白的父亲念了两句“春国送暖百花开，迎春绽金它先来。”，李白的母亲接了第三句“火烧叶林红霞落”，留下了第四句，李白知道这是父母亲要考教自己。李白看着盛开的李花，脱口一句“李花怒放一树白”。他的父亲一听非常满意。忽然想到，李花怒放一树白正好可以成为孩子的名字，于是就取了头尾两个字，李白作为孩子的名字，简单干净。  
④我国地质学家李四光，原名李仲葵。他在 14岁时便以优异的成绩获准公费留学日本，可是他在填写登记表时误将年龄“十四”填入了姓名栏内。由于表格不好换，他急中生智，将“十”字加上几笔改成了“李”字，“四”字却无法再改。但是叫“李四”这名字不好听又没有意义。怎么办呀？他一抬头，看到中堂大匾上有“光被四表”四个大字，这四个大字是形容盛德善行远播四方，于是他便灵机一动，在“李四”后面加上一个“光”字，后来便叫李四光。

# 习作5：我们眼中缤纷的世界

# 关键1：想想印象最深的事物是什么，打算怎样去观察呢？

先想想自己最喜欢且印象最深的是静物、植物、动物，还是哪一处场景？最喜欢它的什么特点呢？准备按什么顺序观察呢？

# 关键 2：细致观察喜欢的事物，写出特点与新发现。

a．要细致观察、抓住事物的突出特点。

围绕自己最喜欢的事物，观察时可以眼看、心想、口尝、手摸、耳听等。也可以用多种观察方法观察，像“横式观察法”，即观察事物的多面性，样子、形状、颜色、大小、轻重、多少、功能、作用等，着力细致观察其突出特点。如，有同学在观察智能手机时，从它的外形、颜色、大小、功能等多方面进行了观察，重点观察了“智能”的特别之处，这样就能获得对事物全面、独特的认识。观察事物时，对于印象特别深的地方，甚至可以用手机或相机拍下来，再进行反复、细致的观察。

b. 观察事物的变化或独特之处，要有新发现。

在本单元的学习中，我们看到“翠鸟会搭船捕鱼，蒲公英颜色会变，杨梅的色、味也会变化”等，这些都是作者观察时的新发现。要想有新发现，可用“纵式观察法”，对事物的发展变化、植物颜色变化、动物生活习性、产品形成过程等，进行连续性观察；也可进行“变式观察”，即变换观察角度、时间、场景等，可在白天、夜晚、晴天、雨雪天等不同天气观察，也可从平视、仰视、俯视等不同角度观察，还可从休息、运动、吃食等不同习性进行观察……“横看成岭侧成峰，远近高低各不同”，这样细致观察，很容易会发现事物的与众不同之处，写出自己的新发现。

# 铜钱草变形记

我房间的窗台上有一盆铜钱草。

铜钱草也叫金钱草。它绿油油的叶子，看上去既像铜钱，又像一片片小小的荷叶。细长的茎，一

节一节地向上延伸，它的生长速度可惊人了，几天的工夫就能蔓延开。

一次偶然的机会，我对它有了更多了解。那次，我因为贪玩忘记了给它浇水，当我想起的时候，铜钱草早就“弯下了腰”，而且有两三片叶子已微微泛黄。我赶紧给它浇水、松土。过了十几分钟，奇迹发生了！铜钱草那细长的叶茎重新立了起来，叶片也像吸足了水的薄海绵，生机勃勃。接下来的几天，我天天按时浇水，不敢有半点马虎，那几片微微泛黄的叶子，竟然又重新绽放了生命的绿色。在阳光的照耀下，一个个圆鼓鼓的“铜钱”在茎上翩翩起舞，像一把把遮阳的小伞，彼此呵护着。

原来，它不仅好看、温柔可人，还有着顽强的生命力呢。我想对它说：“我喜欢你，铜钱草！”

# 17《古诗三首》

1. 《望天门山》是唐代诗人李白所作，诗人李白被称为诗仙。诗中描写的景物有青山、碧水、孤帆。诗中写天门山的山势奇险的两句是：“天门中断楚江开，碧水东流至此回。”其中“楚江”指的是长江中下游，古代流经楚地的部分河段。“两岸青山相对出，孤帆一片日边来。”的意思是：两岸的东梁山和西梁山隔着江水对峙，一叶孤舟从天水相接的远方驶来，远远望去，仿佛来自日边。

2. 《饮湖上初晴后雨》作者是宋代诗人苏轼。诗句“水光潋滟晴方好，山色空蒙雨亦奇。”描写了西湖晴天和雨中的两幅美景。

3. 游览西湖时，导游告诉我们西湖又叫“西子湖”，由此我想到了苏轼的千古名句：欲把西湖比西子，淡妆浓抹总相宜。这句诗的意思是若把西湖比作古代美女西施，淡妆浓抹总是那样合适。赞扬了西湖的美不胜收。

4. 《望洞庭》作者是唐代诗人刘禹锡。“湖光秋月两相和，潭面无风镜未磨。”意思是：月光和水色交融在一起，湖面风平浪静如同未磨的铜镜，平滑光亮。诗中运用比喻的手法，“潭面无风镜未磨”中“镜”是指洞庭湖水面，这个比喻突出了湖面的平静。“白银盘里一青螺”中的“青螺”指的是君山。

# 关键 1：“仔细观察”

同学们，法国雕塑大师罗丹曾说：“生活中不缺少美，只是缺少发现美的眼睛。”在本册第五单元中，我们学习了《搭船的鸟》《金色的草地》两篇课文，读了《我家的小狗》《我爱故乡的杨梅》两篇习作例文，感受了作者细致的观察，体会了细致观察的好处，我们还养成了留心观察的习惯。请你仔细观察一处美景，看看这个地方有些什么，是什么样子的。你还可以试着做一张“观察记录单”，把自己的观察所得记录下来。

# 21《大自然的声音》

# 23《父亲、树林和鸟》

1. 课文通过童年时代父亲和我的对话，展现了父亲善于观察，对鸟的习性十分熟悉，也喜欢树林和鸟，表达了人与自然和谐共处的主题。  
2. “我真高兴，父亲不是猎人”如果父亲是猎人，以父亲对鸟的习性的了解，一定能打到许多鸟。那么，就会有很多鸟儿受到伤害。但父亲不是猎人，所以他当然不会伤害鸟。  
3. “父亲突然站定，朝幽（yōu）深的雾蒙蒙的树林，上上下下地望了又望，用鼻子闻了又闻。”环境描写中“幽深”说明树林很大很安静，不容易发现鸟；“雾蒙蒙”给人一种模糊的感觉，根本看不见鸟。父亲通过听和闻就能在这样的树林里发现鸟，说明父亲对鸟的习性十分了解。  
4. “我茫然地望着凝（níng）神静气像树一般兀（wù）立的父亲。”人物神态描写运用比喻的手法，把父亲比作树，说明父亲的高度专注，倾情投入，父亲仿佛和树融为一体，鸟仿佛就在他的身上一样，也充分表现父亲爱鸟。  
5. “我只闻到有浓浓的苦味的草木气息，没有闻到什么鸟的气味。”“我”闻到的草木的气息，说明“我”不了解鸟。父亲能从这样的气息中闻到鸟的气味，得益于他丰富的亲身体验，可见父亲一直在观察、了解鸟，对鸟的喜爱由来已久。这种对比衬托，突出表现了父亲对鸟的了解。

# 关键 2：梳理意见，提出办法。

这次习作的题目是“我有一个想法”，因此，可以从自己发现的或者同学列举的现象中选择一个写一写。首先，说出自己的想法时内容一定要具体，对问题的危害要说得有理有据。然后，根据你发现的问题梳理出一些意见或建议。梳理时可以通过搜集或查找相关资料，提出自己的思考；可以寻求老师或同学的帮助；也可以请教相关专业人士，对他们做一些采访，积累写作素材。最后，根据搜集的资料进行梳理和整合，寻找解决问题的办法。

# 关键 3：伙伴交流，清晰表达。

伙伴交流时，需要同学们乐于表达，分享自己的看法。你可以先提出你观察到的现象，根据之前积累或搜集的语言材料进行整合，不拘形式地表达自己的见闻和感受。写作时要注意巧妙布局、条理清晰，将自己觉得印象最深刻的内容表达清楚。可以按照“发现问题——梳理问题——提出办法”的思路进行思考、写作。写好后可以读给同伴听，看看同伴能否明白你的想法，再征求同伴对这个问题的一些看法进行修改、完善。

8. 《司马光》这篇课文属于文言文。文言文是中国古代的一种书面语言组成的文章，文字简练，篇幅很短。全文共两句话，但是每个字都代表一个意思，如“戏”便是游戏的意思。句子顺序上也有不同，如“群儿戏于庭”的表达上是人物、事件、地点，而我们以前学的文章的表达顺序一般为人物、地点、事件。

# 习作8：那次玩得真高兴

# 一、抓住重点。

我们把玩法、玩的过程作为重点来写。

# 二、记叙要有顺序。

不管选择什么游戏，都要注意按顺序来比方说：这个游戏的名称，玩儿的规则交代清楚。接着开始游戏，用上表示承接关系的词语或表示时间先后的词语，按游戏进程记叙。最后的结果如何等。

1.动作描写要有选择，要抓住特征性动作或习惯性的动作。

比如：奶奶将右脚绷着，向鞋子口里溜进去，筋脉凸兀的手紧紧抓着门框，将左脚轻轻抬起，紧绷着的脚小心向鞋口一插，又往里扎了扎，接着踩了几踩，让自己的脚更舒适些，满意地出门了。

2.动作描写要具体，要能够描述人物的连续性的动作。

比如：我屏住呼吸，两手弯成弧形，慢慢地靠近它，然后猛扑向前，两手迅速地一扣。哈哈，看 你往哪里跑！

3.要选择恰当的动词，把人物的动作写得准确、具体。

比如：她又看见了一只蝴蝶，便调皮地奔过去，蝴蝶上下飞舞，她目不转睛盯着蝴蝶，终于蝴蝶停在了一朵花上，她躬着背，手中间隔着点空隙，脚尖小心翼翼踮着，汗珠从她的脸上滴落下来，她蹑手蹑脚地走到蝴蝶旁，猛地一弯腰，双手把花朵上的蝴蝶一捧。又小心地把双手露出点缝，把头靠在手缝上看，一不小心让蝴蝶飞出了双手，她又急又气撅着小嘴，双手往腰上一插，但马上又像只小鹿似的蹦跳着追赶另一个目标去了。

4.可以运用比喻、夸张等修辞手法，使动作更形象生动。

比如：枪声一响，两个姑娘如离弦之箭，向前冲去。她们动作协调有力，在冰面上轻快地飞驰，仿佛两只飞燕，在紧贴地面飞翔，你追我赶，互不相让。最后，在教练员和观众热情的加油声中，两人几乎同时像一阵旋风冲过终点。

# 三、过程具体。

叙述经过时，一定要描写人的表现，如人的动作、神态及心理活动等

# 四、感受清楚。

游戏过程中玩得高兴的感受或收获，要清楚的表达出来。

# 那次玩得真高兴

“叮叮叮”上课铃响了，罗老师大步流星地走进教室，对同学们说：“今天，我们来玩一个游戏，好吗？”同学们听了，高兴地跳了起来。

这个游戏叫我来做，你来猜。老师先给我们讲了游戏规则：自由组合，每组三个人，一个同学做动作，另外两个同学猜。哪一组猜对得多，哪一组就赢了。

第一组是：我、林佳妮、何佳昵、李霖莉。李霖莉做动作，我和其他两个同学猜。第一个动作是“吃面条”，她做的动作很象是吃饭，我不假思索地答道：“吃饭”。同学们顿时哄堂大笑，我才知道我回答错了。李霖莉把刚才的动作又表演了一遍，突然，何佳昵灵机一动，说：“吃面条”。老师笑着说：“答对了！”我们都很高兴。接着，第二个抽到的是炒菜，李霖莉一时想不出来怎样做这个动作，就学着她妈妈做菜的样子，先去打煤气灶，何佳昵还没等李霖莉做完动作，就迫不及待地说：“煤气！”惹得全班同学捧腹大笑，老师说时间还没到，让我们继续猜，最后，我们认真地观察了她的动作，结果一致认为是炒菜，老师大声宣布：“答对了，你们真棒！”最后一个动作是骑自行车，林佳妮一会儿就猜出了是骑自行车。我们组全猜对了。

接着，其他组也进行了表演。最后，老师宣布我们队赢了！胜利了！我们四人高兴得一跳三尺高。同学们纷纷为我们鼓掌，老师还给我们发了一本有奖字的作文本呢！站在领奖台上，我很激动，团结协作和不言放弃使我们成功！！！

这个游戏特别有趣，我们玩得真开心。
