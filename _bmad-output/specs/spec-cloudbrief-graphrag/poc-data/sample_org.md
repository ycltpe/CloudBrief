# 云景科技组织架构

## 公司概况
云景科技有限公司成立于 2015 年，总部位于杭州，是一家专注于企业协同 SaaS 的科技公司。

### 高管团队
CEO（首席执行官）：王建国，负责公司整体战略与对外合作。
CTO（首席技术官）：张伟，负责技术研发与基础设施。
CPO（首席产品官）：陈敏，负责产品规划与用户体验。
CFO（首席财务官）：李婷，负责财务、融资与投资者关系。
COO（首席运营官）：刘洋，负责销售、客户成功与日常运营。
