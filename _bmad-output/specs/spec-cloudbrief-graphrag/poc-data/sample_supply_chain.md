# 云景科技供应链伙伴

## 核心供应商

### 云服务器与基础设施
- 星辰云：云景科技的主要公有云服务商，提供 ECS、对象存储、数据库等基础资源。合作始于 2015 年，合同每年续签。
- 银河数据中心：提供混合云托管与专线接入服务。合作始于 2018 年。
- 磐石网络：提供 CDN 与安全防护服务。合作始于 2019 年。

### AI 与模型服务
- 灵犀智算：提供 GPU 算力租赁与大模型微调平台。合作始于 2023 年。
- 百川智能：提供大模型 API 与行业模型授权。合作始于 2024 年。

### 办公与企业服务
- 明道办公：提供企业邮箱、文档协作与审批系统。合作始于 2016 年。
- 锐捷网络：提供企业级网络设备与 Wi-Fi 方案。合作始于 2017 年。

## 下游客户与渠道

### 战略客户
- 华东银行：云景科技企业协同平台的首家金融行业客户，2016 年签约。
- 长江制造：大型制造业集团，2019 年签约，是云景科技在制造业的标杆客户。
- 未来教育：在线教育集团，2020 年签约。

### 渠道伙伴
- 东方国信：云景科技华东区总代理，负责上海、江苏、浙江的客户拓展与实施交付。
- 南方联创：云景科技华南区总代理，负责广东、福建、海南的市场拓展。

## 合作网络
- 星辰云是华东银行的云服务商之一。
- 东方国信同时代理星辰云与云景科技的产品。
- 长江制造同时使用云景科技协同平台与磐石网络 CDN。
- 未来教育与灵犀智算在同一教育 AI 联盟中。
