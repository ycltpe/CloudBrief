问题与答案
Question
报表导出后无法打开怎么办？
Ground Truth
请检查导出格式是否为受支持的格式（CSV/Excel/PDF），尝试重新导出。如仍无法打开，请确认本地办公软件版本是否兼容，或联系技术支持。
System Answer
报表无法打开时，请先确认导出格式是否受支持[^1]。如果格式正确，尝试重新导出一次[^2]。若仍有问题，可能是本地软件版本不兼容。
检索到的片段
导出报表时，请从"报表中心"选择目标报表，点击"导出"按钮。支持的格式包括 CSV、Excel 和 PDF。如导出后无法打开，请检查文件扩展名是否正确。
