# 报表导出与下载指南

CloudBrief 支持将数据报表导出为 CSV、Excel 和 PDF 三种格式，方便客户离线查看。

## 支持的导出格式

- CSV：兼容性好，文件小，适合大数据量。
- Excel（.xlsx）：保留样式和公式，适合需要二次编辑的场景。
- PDF：固定版式，适合打印和归档。

## 导出操作步骤

1. 进入「数据报表」页面。
2. 选择时间范围和筛选条件。
3. 点击右上角「导出」按钮。
4. 选择格式并确认下载。

## 常见问题

### 报表导出后无法打开怎么办？

如果客户反馈导出文件无法打开，请按以下顺序排查：

1. 确认文件是否完整下载。网络中断可能导致文件损坏，建议重新导出。
2. 检查文件格式与本地软件版本是否匹配。例如 Excel 2003 及更早版本默认无法打开 .xlsx 文件，可让客户升级 Office 或改用 CSV 格式。
3. 如果数据量超过 10 万行，Excel 格式可能触发性能限制，建议拆分为多个时间段导出或使用 CSV 格式。
4. 查看是否包含特殊字符导致编码错误。CSV 默认使用 UTF-8，部分旧版 Excel 打开时会乱码，可提示客户使用「数据导入」功能选择 UTF-8 编码。

### 导出为 Excel 格式时报错？

当导出为 Excel 格式失败时，常见原因包括：

- 报表中包含超过 32767 个字符的单元格内容（Excel 单单元格上限）。
- 使用了特殊公式或函数，在生成 Excel 时解析失败。
- 服务器临时文件目录空间不足。

解决方案：先尝试导出 CSV 验证数据完整性；若 CSV 正常，则拆分数据或缩短时间范围后重试 Excel 导出。

## 注意事项

- 导出任务为异步执行，大文件可能需要几分钟。
- 导出链接有效期为 24 小时，过期后需要重新生成。
